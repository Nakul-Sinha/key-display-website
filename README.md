# Key Display Website

## Description

This website allows users to input a single lowercase letter (a-z) and displays the uppercase version of that letter on the screen. It provides a simple and interactive way to visualize key inputs.

## Technologies Used

*   HTML
*   CSS
*   JavaScript

## Folder Structure

```
.
├── index.html
├── style.css
├── script.js
├── README.md
├── .gitignore
└── package.json
```

*   `index.html`: Main HTML file containing the structure of the website, including input field and display area.
*   `style.css`: CSS file for styling the website, including layout, colors, and typography.
*   `script.js`: JavaScript file for handling user input, processing the key, and updating the display area.
*   `README.md`: Project documentation.
*   `.gitignore`: Specifies intentionally untracked files that Git should ignore.
*   `package.json`: Contains metadata about the project and manages dependencies.

## Setup and Usage

1.  Clone the repository to your local machine.
2.  Open `index.html` in your web browser.
3.  Enter a lowercase letter (a-z) in the input field.
4.  Click the "Display" button to see the uppercase version of the letter displayed.

## Features

*   Simple and intuitive user interface.
*   Input validation to ensure only lowercase letters are accepted.
*   Displays the uppercase version of the input.
*   Clear input field after displaying the key.
*   Fade-in animation for the displayed key.
*   Enter key functionality to trigger display.
